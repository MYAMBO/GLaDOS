# Sliece for VS Code

This extension adds syntax highlighting for the Sliece programming language.

## Features

- Syntax highlighting for `.lie` files
- Supports:
  - `func`, `define`, `$` for function definitions
  - `//` and `/* ... */` comments
  - Types: Int8, Int16, Int32, etc.
  - Operators: `->`, `=>`, `==`, etc.

## Usage

1. Open any `.lie` file in VS Code.
2. Enjoy syntax highlighting for Sliece!
